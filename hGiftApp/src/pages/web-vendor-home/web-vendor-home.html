VENDOR HOME

Using the Hybrid Gifting app users can use Gift Creation Kits to create personalised digital-physical gifts. Gift Creation Kits contain all the material to create a gift (or turn an existing physical gift into a hybrid digital-physical gift). This includes NFC tags and QR codes with preconfigured data and media provided by the manufacturer.

As a museum, gift shop, manufacture, or other vendor you can use this page to create this preconfigured data, Gift URIs, and upload this media, Default Content, to create your Gift Creation Kits.
